# Peta Tematik Persebaran Ruko di Kelurahan Sukun

This is a WebGIS project that displays the thematic distribution of shophouses (ruko) in the Sukun sub-district, Malang City. This project was built using QGIS with the qgis2web plugin and exported to Leaflet.

## Demo

A live demo of the project is available [here](https://fiyuudump.github.io/gis-ruko-sukun/).

## Features

- **Interactive Map**: An interactive map displaying the distribution of shophouses.
- **Thematic Layers**: Includes various thematic layers such as residential areas, open green spaces, and roads.
- **Search Functionality**: Users can search for specific shophouse locations.
- **Responsive Design**: The user interface is designed to be responsive and work on various devices.
- **Landing Page**: A modern landing page with animations and a project overview.

## Tech Stack

- **QGIS**: Used for creating the initial map and data layers.
- **qgis2web**: A QGIS plugin used to export the project to a web format.
- **Leaflet**: A JavaScript library for interactive maps.
- **Tailwind CSS**: For styling the user interface.
- **GSAP (GreenSock Animation Platform)**: For animations on the landing page.
- **GitHub Pages**: For hosting the web application.

## Setup and Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Fiyuudump/gis-ruko-sukun.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd gis-ruko-sukun
    ```
3.  **Open `index.html` in your browser to view the landing page.**

## Deployment

This project is automatically deployed to GitHub Pages using GitHub Actions. The workflow is defined in `.github/workflows/deploy.yml`. Any push to the `main` branch will trigger a new deployment.
